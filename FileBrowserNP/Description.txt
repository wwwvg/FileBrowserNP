+	Приложение состоит из двух вертикальных панелей с регулируемым разделителем между ними. 

+	В левой части находится браузер на основе списка (ListView в режиме GridView) для просмотра дисков, 
	папок и файлов локального компьютера, аналогично программе Total Commander. 
+	В списке предусмотрен элемент для возврата в родительскую папку. 
+	В правой, содержимое выбранного файла. Сделан вывод <<ТЕКСТОВЫХ ФАЙЛОВ>> и <<КАРТИНОК>> (jpg, bmp, png), 
	а <<ДРУГИЕ ФАЙЛЫ>> выводятся в HEX формате по 16 чисел в строке, разделенных пробелом.						
+	По нажатию на клавишу Enter реализован вход в дочерние папки или выход в родительскую.
+	В тулбаре, доступны команды создания новой папки и удаления выбранного элемента с подтверждением через диалог. 
+	Для новой папки предусмотрен ввод имени в диалоге. 
	После выполнения команды создания или удаления, список отображает эти изменения.
+	В статусбаре сделан вывод полного пути к выбранному элементу, а также время изменения и размер для файла.



1)+  Отказаться от использования Prism и других MVVM библиотек.

2)+  Добавить диски в общий список.

3)+  Использовать различные типы для элементов списка.

4)+	 Использовать DataTemplate для вывода элементов и их контента.

5)+  Исправить ошибку с удалением открывавшихся файлов.

6)+  Открывать контент при выборе элемента (selection).

7)+  Исправить установку фокуса при возврате в родительскую папку, если она не находится в области видимости.

8)+  Исправить реакцию на двойной щелчок. Сделать только для элемента, а не для всего контрола ListView.
